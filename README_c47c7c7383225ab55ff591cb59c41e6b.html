Add JBoss in Eclipse

  1. Install JBoss plugin for Eclipse
  2. In Eclipse: Window -> Show View -> Servers
  3. In Servers: Right click -> New -> Server
  4. Add server runtime environment
  5. Set JRE for server runtime environment
